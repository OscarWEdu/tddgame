# Description

The TDDGame project is a version of the traditional game RISK with a few notable changes, most importantly, victory and defeat in this game is determined via what we like to call "Typing Challenges" who has the best accuracy and speed. TDDGame is a project built on the principles of Test Driven Development, and is intended to let our contributors test out the application TDD principles.

# How to Install and Run

- Install Node.js and .NET 10.
- Run npm install to install all packages and dependencies.
- Add a correctly configured backend/db-config.json file, the structure is shown further below.
- Launch application with npm run dev.
- Open the website at: http://localhost:5173

# db-config.json example

```
{
  "host": "",
  "port": ,
  "username": "",
  "password": "",
  "database": "",
  "createTablesIfNotExist": true,
  "seedDataIfEmpty": true,
  "aiAccessToken": "",
  "smtpServer": "",
  "smtpPort": ,
  "emailUsername": "",
  "emailPassword": ""
}
```
